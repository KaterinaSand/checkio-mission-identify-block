You are given 4 numbers (set of integer).
These number represents a block. You have to identify kind of the block.

It is possible that the block rotated (0, 90, 180 and 270 degree). If it isn't a block, Then return None.

Input: 4 numbers (set of integer)

Output: kind of the block (an alphabet or None)

Example: 

identify_block({1, 2, 3, 4}) == 'I'
identify_block({1, 2, 3, 6}) == 'T'
identify_block({1, 5, 6, 10}) == 'S'

Precondition:

  • len(input) == 4
  • output in ('I', 'J', 'L', 'O', 'S', 'T', 'Z', None)
